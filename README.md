# AutoShkolla Pro — Sistem Profesional

## Si ta nisni lokalisht

```bash
npm install
npm run dev
```

Hapni: http://localhost:5173
Login: admin@autoshkolla.com / admin123

## Si ta hostoni në GitHub Pages (1 click)

1. Krijoni repo në GitHub
2. Ngarkoni gjithçka (ose vetëm folderin `dist/`)
3. Settings → Pages → Deploy from branch → `main` → `/docs`
4. Ose: zhvendosni `dist/` → emërojeni `docs/` → push

## Si ta hostoni në Netlify (drag & drop)

1. Hapni: https://netlify.com/drop
2. Tërhiqni folderin `dist/` direkt në faqe
3. URL gati brenda 30 sekondave

## Teknologjitë

- React 18 + React Router v6
- Vite (build tool)
- TailwindCSS
- Recharts (grafika)
- localStorage (persistence)
- xlsx (Excel export)
